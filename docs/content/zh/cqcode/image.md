## 辅助函数

```js
const { image } = require('cqbot')
image(/*参数*/)
```

## 参数

| 字段名 | 数据类型 | 默认值 | 说明 |
| :---: | :---: | :---: | :---: |
| `file` | string | | 图片路径 |

## 示例

```js
image('https://baidu.com/1.jpg')
```
